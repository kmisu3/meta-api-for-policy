プライバシーポリシー

⚠️ テスト環境について

本システムは開発・テスト用途で構築されており、実際の商用利用は行っておりません。

1. 収集する情報

当システムでは、以下の情報を収集する場合があります：

  • Instagram アカウント情報（ユーザー名、プロフィール情報）
  • 受信したメッセージの内容
  • 自動返信の履歴
  • システム利用時のアクセスログ

2. 情報の利用目的

収集した情報は以下の目的で利用します：

  • Instagram メッセージの自動返信機能の提供
  • システムの動作確認・改善
  • 技術的な問題の解決・サポート
  • テスト・開発目的での分析

3. 情報の管理・保護

  • 収集した情報は適切なセキュリティ対策のもとで管理します
  • 不要になった情報は適切に削除します
  • 第三者への提供は行いません（法令に基づく場合を除く）
  • テスト環境のため、データは予告なく削除される場合があります

4. Instagram API の利用について

当システムは Instagram Basic Display API および Instagram Messaging API を利用しており、 Meta（旧Facebook）のデータポリシーに準拠しています。

5. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、システム管理者までご連絡ください。
